# Treasure Ireland

## Description

 Treasure Ireland is a comprehensive e-commerce platform. It allows users to browse products, add them to a cart, and proceed to checkout. Admin users have the ability to add new products to the platform and check all orders.

## Live Demo

The application is deployed and can be accessed at [treasure-ireland.vercel.app](https://treasure-ireland.vercel.app/)

## Features

- User Authentication: Users can register and log in to the platform. 
- Product Browsing: Users can browse through the list of available products. 
- Shopping Cart: Users can add products to their shopping cart. 
- Checkout: Users can proceed to checkout and place their orders. 
- Admin Features: Admin users can add new products to the platform and check all orders made from client. Also they are able to see a summary of the business.
